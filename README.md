# Steep

**Your repo's fortune, steeped in truth.**

Paste a GitHub repo. Get a dramatic, wildly specific, and entirely useless tea leaf reading of your codebase.

It is a teapot. It cannot brew coffee.

## Setup

```bash
npm install
vercel dev
```

Set `GEMINI_API_KEY` in your Vercel environment variables.

## Stack

Vanilla HTML/CSS/JS · Vercel serverless · GitHub REST API · Google Gemini API
